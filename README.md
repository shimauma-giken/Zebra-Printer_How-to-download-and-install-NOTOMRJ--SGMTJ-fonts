 # NOTOMRJ/SGMTJのインストール方法
 
 1. Win 1x パソコンに下記をインストールする。インストーラはプリンタのサポートページからダウンロード可能。
  - Zebra Setup Utilities for Windows  
  - Windows ドライバ V8  
 2. Zebra Setup Utiities を使用して、各フォントファイルをプリンタに送付する。
